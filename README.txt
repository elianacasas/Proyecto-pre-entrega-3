# Tercera-pre-entrega-Casas Este repositorio contiene los archivos de todo un proyecto de pagina web terminado y listo para ser ejecutado. La carpeta App1 es la destinada a todo lo relacionado con la aplicacion, los archivos html, css y py que la conforman. Los nombres de los html, las funciones, los models, forms y urls indican su funcionalidad. Se trata de una pagina web en la que se pueden ver, buscar y agregar cafeterías que esten en Montevideo. La pagina contiene una seccion en la que se pueden encontrar todas las cafeterias subidas hasta el momento por usuarios, asi como tambien la opcion de agregar locales o buscar por direccion. Además existe la opcion de suscribirse y realizar una reseña sobre la misma. Mismo en la pagina de inicio de pueden ver todas las opciones botones que redirigen a cada seccion.
